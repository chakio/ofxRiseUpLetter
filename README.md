ofxRiseUpLetter
====
* パーティクルで文字の描画
![result](https://github.com/chakio/ofxRiseUpLetter/blob/master/media/font.gif) 
## Description
文字の輪郭情報を元にパーティクルで文字の描画をしてみました。
ランダムに文字が発生します。

## Requirement
* Ubuntu16.04
* openFrameworks v0.10.0
* visualstudio code
